Un formulario es un documento, físico o digital, elaborado para que un usuario
introduzca datos estructurados (nombres, apellidos, dirección, fecha , etc.) en las zonas
correspondientes, para ser almacenados y procesados posteriormente.

Esto ayuda a que diferentes instancias registren datos personales de quien los llena para posteriormente ser acreedor al servicio solicitado, siempre y cuando, los datos sean llenados correctamente.
